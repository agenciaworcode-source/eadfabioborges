# 1. Contexto & Stack

**Plataforma EAD premium em código puro para a Mentoria Fábio Borges — stack mínima, VPS própria, 7 semanas.**

## Por que código puro e não WordPress?

Posicionamento premium exige controle total de design e performance. Elementor é lento e genérico. Com Next.js + Supabase: 100% personalizado, sem licenças mensais, sem conflitos de plugins, deploy em VPS própria.

## Números do Projeto

- 7 Semanas
- 280h Total estimado
- 9 Cursos ativos
- 4 Planos de mentoria

## Stack Técnica (mínima, segura, sem vendor lock-in)

| Camada              | Tecnologias                                                                                                        |
| ------------------- | ------------------------------------------------------------------------------------------------------------------ |
| Frontend            | Next.js 14 App Router · React 18 · TypeScript · Tailwind CSS · Poppins                                             |
| Backend & Banco     | Supabase PostgreSQL · Supabase Auth · Row Level Security · Supabase Storage · Edge Functions                       |
| Pagamentos          | Stripe Checkout · Stripe Webhooks · Stripe Subscriptions · MercadoPago (PIX)                                       |
| Email / Vídeo / PDF | Resend (transacional) · React Email (templates) · Vimeo Embed API · pdf-lib (certificados) · qrcode (npm)          |
| Infraestrutura      | VPS 4 vCPU / 8 GB RAM · Ubuntu 24.04 LTS · Nginx · PM2 (cluster mode) · Let's Encrypt SSL · GitHub Actions (CI/CD) |
