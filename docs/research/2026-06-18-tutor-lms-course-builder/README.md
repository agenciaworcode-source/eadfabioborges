## TL;DR

O Tutor LMS é um plugin WordPress LMS com mais de 100.000 sites ativos. Seu diferencial central é o **Course Builder visual integrado diretamente no wp-admin**, com abas organizadas (Basic, Curriculum, Additional Settings), drag-and-drop nativo de tópicos e lições, e **8 tipos de pergunta de quiz** — um dos mais completos entre LMS WordPress. A versão Pro desbloqueia add-ons críticos como Certificate, Content Drip, Live Classes (Zoom/Google Meet), Assignments e Course Preview.
