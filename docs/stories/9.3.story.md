## Story

**As a** aluno autenticado,
**I want** acessar `/dashboard/cursos` e ver todos os cursos em que estou matriculado com progresso, status e acesso direto ao conteúdo,
**so that** eu tenha uma visão consolidada de todas as minhas matrículas, diferente da visão "continue de onde parou" da home do dashboard.

## Business Value

O Sidebar da plataforma já linka para `/dashboard/cursos`, mas a página não existe — retorna 404. Qualquer aluno que clicar em "Meus cursos" no menu encontra um erro. Essa página é o segundo item de navegação mais acessado após a home do dashboard. Fechar esse 404 é crítico para a experiência do aluno antes do go-live.

## Technical Notes

### Padrão de query — reaproveitar lógica do dashboard

A lógica de busca de matrículas + cálculo de progresso já existe em `src/app/(dashboard)/dashboard/page.tsx`. Reutilizar a mesma lógica:

```typescript
// Matrículas + cursos
const { data: enrollments } = await supabase
  .from('enrollments')
  .select('*, courses(*)')
  .eq('user_id', user.id)
  .in('status', ['active', 'completed'])

// Progresso por curso — igual ao dashboard/page.tsx
// Para cada enrollment: buscar moduleIds → lessonIds → lesson_progress count
```

> **Decisão de design:** a query de progresso no dashboard usa `Promise.all()` com queries individuais por curso — aceitável para um aluno típico (<20 cursos). Manter o mesmo padrão para consistência; não otimizar prematuramente.

### Estrutura de card do curso

```tsx
<Link key={enrollment.id} href={`/dashboard/curso/${course.id}`} className="ccard">
  {/* Thumbnail ou fallback */}
  <div className={`thumb ${getThumbVariant(idx)}`}>
    {course.thumbnail_url
      ? <img src={course.thumbnail_url} alt={course.title} style={{ width:'100%', height:'100%', objectFit:'cover' }} />
      : <CourseIcon />
    }
  </div>
  <div className="body">
    <h3>{course.title}</h3>
    <div className="progress-bar" ...>
      <span style={{ width: `${pct}%` }} />
    </div>
    <div className="foot">
      <span>{pct}% concluído</span>
      {isComplete && <span className="badge green">Concluído</span>}
    </div>
  </div>
</Link>
```
