## Story

**As a** Fábio Borges (administrador da plataforma),
**I want** uma gestão de alunos e matrículas completa no painel admin,
**so that** eu possa ver o perfil completo de cada aluno (cursos matriculados, progresso, plano), matricular/desmatricular com poucos cliques, ter uma aba dedicada de matrículas filtrável por curso, e exportar dados para análise externa — com experiência equivalente a plataformas LMS profissionais.

## Business Value

Hoje o modal de aluno mostra apenas 4 números estáticos e os botões "Editar aluno" / "Enviar mensagem" não fazem nada. Não existe forma de ver em quais cursos o aluno está, qual é o progresso, nem gerenciar matrículas pelo perfil do aluno. A matrícula manual existe mas é um formulário solto sem contexto. Isso deixa o admin "cego" sobre o status real de cada aluno.

### Exportação CSV (client-side)

```typescript
function exportToCSV(rows: EnrollmentRow[], filename: string) {
  const headers = ['Aluno', 'Email', 'Curso', 'Data Matrícula', 'Progresso', 'Status']
  const lines = rows.map((r) => [
    r.user.name,
    r.user.email,
    r.course.title,
    new Date(r.enrolled_at).toLocaleDateString('pt-BR'),
    `${r.progress}%`,
    r.status,
  ])
  const csv = [headers, ...lines].map((row) => row.map((v) => `"${v}"`).join(',')).join('\n')
  const blob = new Blob(['\uFEFF' + csv], { type: 'text/csv;charset=utf-8;' })
  const url = URL.createObjectURL(blob)
  const a = document.createElement('a')
  a.href = url
  a.download = filename
  a.click()
  URL.revokeObjectURL(url)
}
// '\uFEFF' = BOM para Excel abrir UTF-8 corretamente
```
