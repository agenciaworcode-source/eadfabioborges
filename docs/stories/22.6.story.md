## Story

**As a** Fabio Borges (administrador da plataforma),
**I want** uma area dedicada para gerenciar avaliacoes, provas finais e quizzes por aula,
**so that** eu consiga configurar provas por curso sem depender de botoes escondidos dentro do Course Builder.

## Business Value

O builder atual deixa avaliacoes escondidas dentro do editor de cursos e dentro de cada aula. Isso prejudica a operacao diaria: Fabio nao enxerga quais cursos tem prova final, quais quizzes existem por aula, nem onde configurar rapidamente uma avaliacao. Uma central dedicada reduz friccao e torna avaliacao uma entidade de gestao pedagogica.

## Scope

### IN

- Nova rota `/admin/avaliacoes`
- Link "Avaliacoes" no sidebar admin
- Tela com visao de **Provas finais** por curso
- Tela com visao de **Quizzes por aula**
- Acao para criar/editar prova final vinculada a um curso
- Reuso do `QuizBuilder` existente para manter CRUD de questoes
- Editor de curso passa a mostrar link para a Central de Avaliacoes em vez de ser o fluxo principal

### OUT

- Player do aluno para prova final
- Correcao manual de dissertativas
- Relatorios avancados de desempenho
- Timer, feedback mode e novos tipos de questao do Epic 22.5

## Acceptance Criteria

- **AC-1:** `/admin/avaliacoes` carrega com protecao do layout admin existente
- **AC-2:** Sidebar admin exibe link "Avaliacoes"
- **AC-3:** Aba/visao "Provas finais" lista todos os cursos com status da prova final
- **AC-4:** Cada curso mostra: titulo, status publicado/rascunho do curso, total de questoes da prova final, nota minima e tentativas
- **AC-5:** Um botao "Configurar prova" abre o builder da prova final do curso
- **AC-6:** Aba/visao "Quizzes por aula" lista quizzes existentes por curso > modulo > aula
- **AC-7:** Editor de curso mantem um resumo/link para `/admin/avaliacoes?courseId={id}` sem duplicar a experiencia principal
- **AC-8:** `npm run lint`, `npm run typecheck` e `npm test` passam
