## Story

**As a** Fábio Borges (administrador da plataforma),
**I want** um painel admin completo para gerenciar alunos, cursos e visualizar relatórios,
**so that** eu possa operar toda a plataforma sem precisar acessar o banco de dados diretamente.

## Business Value

Sem o painel admin, Fábio precisa usar o Supabase Studio para qualquer operação: matricular aluno, editar título de curso, ver quantos alunos concluíram. Isso é um bloqueador operacional direto — o painel admin é a interface de controle do negócio. Com ele, Fábio tem autonomia total sobre a plataforma via UI.

### Donut chart (conic-gradient CSS puro)

```typescript
// Distribuição por plano: Free, Prata, Ouro, Diamante
// Calcular offset acumulado para conic-gradient
function buildDonut(counts: Record<string, number>) {
  const total = Object.values(counts).reduce((a, b) => a + b, 0)
  let acc = 0
  return Object.entries(counts).map(([plan, n]) => {
    const pct = (n / total) * 100
    const segment = { plan, start: acc, end: acc + pct }
    acc += pct
    return segment
  })
}
// background: conic-gradient(color1 0% start1%, color2 start1% start2%, ...)
```
